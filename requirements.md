# REQUIREMENTS

### BASICS

- [ ] Multiple Tools - Rectangle , Circle , Pen etc.
- [ ] Movable Objects
- [ ] Resizable Objects
- [ ] Text insertion
- [ ] Zoom In/Out

### Objects Props

- X coord : Number
- Y coord : Number
- Type : String
- Width : Number
- Height : Number
- StrokeWidth : Number
- Stroke Style (color) : String
- Fill : Boolean
- Rotation : Number
- Updated : DateTime
